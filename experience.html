Personal Seat Companion

Coding

Jan 2016 - Present

This is a new project based on an idea I had during the final exam period. I had a hard time finding a single seat in school libraries. The DP Library of UWaterloo has 10 floors; However, I did not even know which button to press when I was in the library elevator because I didn't know which floor had empty seat.

So I was thinking what if there is an application to help UWaterloo students find a seat?

The prerequisite is there should be a sensor for every seat. If there is not, student has to register on the app himself (We'd better not depend on our clients for data).

Once we get the status of every seat, our algorithm will provide recommended seats based on client's request. Basic requests include # of seats required, # of plugs required, lighting, distance to current position, etc. After linking with fxxxbook, student can see the study places of his friends.

I just started this project with my partner. Hopefully it can be finished this winter. It is written in C++.

Sudoku Solver

Coding

Dec 2014 - Jan 2015

Yes, this project is based on my CS assignment. However, it means much more to me than just an assignment.

Sudoku means a lot to me. I'm a long-term Sudoku lover. I was founder and president of my high school Sudoku Club. This project was one of the qestions in the last assignment of my very first CS course. Upon finishing the code for the first time, it can gave answers to simple Sudoku problems in 30 sec but for hard problems, it continued running after 1 hr. I was upset. Though it was ok if our solution could not solve all problems, I decided to find out why my solution failed difficult Sudoku questions.

I went to Dave's office hours. It turned out I wrote some extra code that slowed down my program. The extra code was to look for cells that have few options of numbers. It is a technique I use a lot when I'm solving Sudoku. Since I'm a human and sensitive to numbers, I can find and fill in the cell that only fits one number. However, this piece of code is not for computer programs. After removing it, my program can solve any Sudoku problem in 10 sec.

I've spent some time on both this question and the other one about Solitaire. For both questions, I had already met the requirements of assignments but I managed to improve my programs. To be honest, I had not declared my major back then. The moment I saw my Sudoku program gave the perfect answer after I poured my efforts was the moment I fell in love with CS.
